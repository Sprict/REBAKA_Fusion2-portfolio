# ふと思いついたアイデアのメモ集

## ゲーム仕様

### 世界観について

- 地下坑道に金稼ぎに送り出された恐竜たち（仮）
- トロッコに洞窟で見つけたお宝を入れていって脱出（←トラップとトロッコが両立できるのか要検証)

### プレイヤー死亡時の挙動

- プレイヤー死亡後一定時間後にパトラッシュさながらに点からドラゴン天使（?）が下りてきて、死んだプレイヤーを連れ去ろうとする
- 仲間が点に召されないように地面に引っ張ることで昇天を防いで体力１で復活させることができる
- 昇天を防いでいる時、助ける仲間が一人ならそいつが少し地面から浮いて足をばたつかせているシーンがはたから見ていて面白そう

### プレイヤーの体力表示について

- できるだけ文字などのUIを避けたい。UIはできるだけインゲームに入れたい
- 【提案】現状のプレイヤーの合計10本のたてがみを残りHPに見立ててHPが10%減るごとに下の方から一本ずつ消滅していく（消滅エフェクトは分かり易くすべき)

### フレンドリーファイアはあり

- 仲間のせいで死ぬのも笑いの種になるから
- プレイヤーの性格によっては人狼したい人もいると思うから。でも私は人狼が嫌いだしあくまでCOOPゲームという設計なのでゲームシステムには組み込まない。したい人はどうぞご自由にという感じ。
- FFありにするかわりにホストは参加プレイヤーのキックやブロックをできるようにした方がいいかも
