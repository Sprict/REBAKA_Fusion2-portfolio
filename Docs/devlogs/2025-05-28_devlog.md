## 1. Photon Fusionの問題調査と解決

### NetworkInputData関連のエラー

- 問題: Hostボタンをクリック時にAssertException: GetTypeKey for MyFolder.Scripts.Network.NetworkInputDataエラーが発生
- 原因: NetworkInputDataクラスに[System.Serializable]属性が必要だった
- 解決: NetworkInputData.csに[System.Serializable]属性を追加

### NetworkButtons関連のエラー

- 問題: AssertException: Exception of type 'Fusion.AssertException' was thrown.エラーが発生
- 原因: ButtonUtils.csでのボタン定数定義がNetworkButtonsの制限（0～31の整数値）を超えていた
- 解決: ButtonUtils.csのボタン定数をビット演算子による定義から単純な整数値（0～6）に変更

```csharp
// 修正前
public const byte BUTTON_MOUSE0 = 0x01;
public const byte BUTTON_MOUSE1 = 0x02;
public const byte BUTTON_RIGHTPUNCH = 0x04;
public const byte BUTTON_LEFTPUNCH = 0x08;
public const byte BUTTON_JUMP = 0x10;
public const byte BUTTON_DASH = 0x20;
public const byte BUTTON_CROUCH = 0x40;  // 問題の原因（64は31を超える）

// 修正後
public const byte BUTTON_MOUSE0 = 0;
public const byte BUTTON_MOUSE1 = 1;
public const byte BUTTON_RIGHTPUNCH = 2;
public const byte BUTTON_LEFTPUNCH = 3;
public const byte BUTTON_JUMP = 4;
public const byte BUTTON_DASH = 5;
public const byte BUTTON_CROUCH = 6;
```
