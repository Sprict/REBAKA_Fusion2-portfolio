# REBAKA_Fusion2 FAQ

プロジェクトについてユーザーから繰り返し問われる疑問と、その回答を永続化する場所。

## 使い方（Claude 向け）

- 同じ質問が **2 回以上** 出たら、ここに追記する
- 調査結果の要約を残す。生のコード貼り付けではなく「どこを見ればよいか」を記述
- 古くなった回答は更新する(削除せず日付付きで訂正)

## 構成

質問はカテゴリで分類する:

- **アーキテクチャ**: コード構成、責務分離、ファイル関係
- **物理演算**: ラグドール、PID、APR 関連
- **ネットワーク**: Photon Fusion、同期、入力
- **ワークフロー**: git, devlog, uLoopMCP の使い方
- **その他**
